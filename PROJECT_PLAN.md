# 🍋 루틴핏 (RoutineFit) - 프로젝트 설계서

## 앱 컨셉
> "매일 작은 루틴이 건강한 삶을 만든다"
현대인의 건강한 생활패턴을 만들어주는 루틴 기록 & 커뮤니티 앱

---

## 핵심 기능 구성 (메인 vs 서브)

### 🌟 메인 기능
| 기능 | 설명 |
|------|------|
| 오늘의 루틴 체크리스트 | 아침/직장/저녁 루틴을 체크하며 달성률 확인 |
| 건강 커뮤니티 | 운동완료, 식단인증, 건강팁 공유 |
| 오늘의 경제+건강 뉴스 | AI가 요약한 뉴스 3분 읽기 |

### 📌 서브 기능
| 기능 | 설명 |
|------|------|
| 혈당 관리 가이드 | 당뇨 예방, 혈당 스파이크 방지 팁 |
| 앉아서 하는 스트레칭 | 직장인용 의자 스트레칭 가이드 |
| 수분 섭취 알림 | 물 마시기 리마인더 |
| 건강 점수 (추후) | 루틴 달성률 기반 건강 점수 |

---

## 🗺️ 페이지 구조
```
/ (홈)
├── 오늘 날짜 + 달성률 헤더
├── 아침 루틴 체크리스트
├── 직장 루틴 체크리스트
├── 오늘의 건강 팁
└── 커뮤니티 미리보기

/routines (루틴)
├── 아침 루틴 (물, 스트레칭, 뉴스, 식사)
├── 직장 루틴 (의자 스트레칭, 눈 휴식, 수분)
├── 저녁 루틴 (저녁 산책, 스트레칭, 금식)
└── 건강 관리 (혈당, 당뇨 예방)

/news (뉴스)
├── 오늘의 경제뉴스 요약
├── 건강 정보
└── 혈당 관리 팁

/community (커뮤니티)
├── 전체 피드
├── 운동 완료 인증
├── 식단 인증
└── 건강 팁 공유

/auth/login (로그인)
/auth/signup (회원가입)
```

## 🛠️ 기술 스택
| 구분 | 기술 | 이유 |
|------|------|------|
| 프론트엔드 | Next.js 14 + TypeScript | 빠른 개발, SEO 최적화 |
| 스타일링 | Tailwind CSS | 레몬옐로우 테마 빠른 적용 |
| 데이터베이스 | Supabase (Phase 2) | 무료, 쉬운 설정 |
| 인증 | Supabase Auth (Phase 2) | 소셜 로그인 포함 |
| AI 콘텐츠 | Claude API (Phase 3) | 뉴스 요약, 건강 글 생성 |
| 배포 | Vercel | 무료, GitHub 연동 자동 배포 |

## 📱 유튜브 & 인스타 성장 전략

### 유튜브
- **콘텐츠**: "5분 아침 스트레칭", "혈당 안 오르는 식사법", "직장인 루틴"
- **앱 연동**: 유튜브 영상을 앱 내 루틴 가이드에 임베드
- **CTA**: "앱에서 루틴 체크하기" → 앱 다운로드 유도

### 인스타그램
- **콘텐츠**: 오늘의 건강팁 카드뉴스 (앱 디자인과 동일 스타일)
- **공유 기능**: 앱에서 루틴 완료 시 인스타 공유 버튼
- **챌린지**: "30일 루틴 챌린지" 해시태그 이벤트

### 피드백 수집
- 앱 내 피드백 버튼 (간단한 설문)
- 인스타 DM / 댓글
- 카카오 오픈채팅방 운영
