# 🍋 루틴핏 (RoutineFit)

> 매일 작은 루틴이 건강한 삶을 만든다

## 🚀 로컬에서 실행하기

### 1단계: Node.js 설치 (처음 한 번만)
- https://nodejs.org 에서 LTS 버전 다운로드 & 설치

### 2단계: 패키지 설치
터미널(명령 프롬프트)을 이 폴더에서 열고:
```bash
npm install
```

### 3단계: 개발 서버 실행
```bash
npm run dev
```

### 4단계: 브라우저에서 확인
- http://localhost:3000 접속
- 모바일 테스트: 크롬 개발자도구(F12) → 모바일 아이콘 클릭

---

## 📋 현재 구현된 기능 (Phase 1)
